Welcome to Full Stack App

A static prototype simulating employee systems workflows. Experience:

  • User Registration with verification state workflows.
  • Role-Based Navigation Guards (Administrative vs Standard access limits).
  • Full dynamic CRUD capabilities for Employees, Departments, and Accounts.
  • Hardware request forms built with dynamic checklist line items.
Prototype Sandbox: All data modifications are committed to localStorage. Clear your browser storage or click reset in lists to restore seed defaults.
